# Introduction

<h2 id="overview">Overview</h2>

React Block Editor is a versatile block-style editor designed to facilitate the creation
of rich-text documents within React applications. Originally developed as a fundamental
feature for the [FanStop](https://fanstop.jackdunn.info) project, the React Block Editor
has evolved into a standalone component and library, ready to empower developers in crafting dynamic and engaging interfaces.

The React Block Editor introduces a novel approach to content creation by leveraging contenteditable divs organized within containers known as blocks. This architecture enables users to seamlessly manipulate and structure text, images, and multimedia elements, empowering them to generate large, intricate documents with ease.

<h2 id="key-features">Key Features</h2>

### Block API

Utilize the Block API to create custom block types tailored to specific content requirements. With access to the Block API, developers can define their own block structures, behaviors, and functionalities, enabling endless possibilities for content creation.

### Theming

Customize the appearance of components to match the visual style of your application using theming capabilities. With flexible theming options, developers can effortlessly integrate the React Block Editor into their projects while ensuring a cohesive and branded user experience.

<h2 id="getting-started">Getting Started</h2>

To integrate the React Block Editor into your project, simply install the package
via npm or yarn and import it into your React components. Refer to the installation
and usage guides for detailed instructions on incorporating the editor into your
application.
